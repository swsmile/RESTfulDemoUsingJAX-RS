# RESTfulDemoUsingJAX-RS
A simple RESTful demo based on JAX-RS (Java API for RESTful Web Services)

# Usage
### Run 
mvn exec:java

### WADL: http://localhost:8080/myapp/application.wadl

### Visit: http://localhost:8080/myapp/myresource
